# Food Tracker

##### Epicodus exercise using angular2, 3/25/16

##### Brad Pizzimenti

## Description
A simple Angular 2 app for adding foods, calories, and descriptions to a list.


## Setup

Clone this repository:
```
$ cd ~/Desktop
$ git clone https://github.com/pizzimenti/js10
$ cd js10
```

Install dependencies:
```
$ npm install
$ bower install
```

Build and Serve:
```
$ gulp build
$ gulp serve
```
